# משחק אותיות ומילים (Hebrew Alef-Bet)

אפליקציית Web פשוטה לילדים ללימוד אותיות ומילים בעברית.

## איך להפעיל מקומית
פתחו את `index.html` בדפדפן.

## GitHub Pages
1. העלו את הקבצים לריפו (root).
2. Settings → Pages → Deploy from a branch → main / root.
3. פתחו את הלינק שנוצר.

## התקנה כאפליקציה (PWA)
באייפון/אייפד: Share → Add to Home Screen.

## הרחבות אפשריות
- ניקוד (מילים עם ניקוד)
- רמות לפי אותיות ספציפיות
- אוצר מילים לפי נושאים
- תמונות לכל מילה
